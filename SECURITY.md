# Security Policy

## Educational Scope

VulnShop-Lab intentionally contains vulnerabilities for local educational use. Findings related to the intentional vulnerable behavior are expected and should not be reported as security issues.

## Responsible Use

Use this project only in:

- Local lab environments
- Authorized training environments
- Classroom or workshop settings with permission

Do not deploy the vulnerable app to the public Internet.

## Reporting Real Issues

If you discover a non-intentional issue in the secure app, documentation, build pipeline, or report generator, please open a GitHub issue with:

- A clear description
- Reproduction steps
- Expected behavior
- Actual behavior
- Suggested fix, if available
