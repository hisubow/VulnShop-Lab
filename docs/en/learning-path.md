# Learning Path - English

## Phase 1: Run the lab

- Install Docker Desktop.
- Run `docker compose up --build`.
- Open both apps on ports 5000 and 5001.

## Phase 2: Explore the web app

- Log in with demo accounts.
- Try Search, Feedback, Profile, and Admin.
- Observe Docker logs.

## Phase 3: Complete challenges

- Read `docs/en/challenges.md`.
- Test each case on the vulnerable app.
- Repeat on the secure app.

## Phase 4: Collect evidence

- Use `docs/en/evidence-checklist.md`.
- Record endpoint, account, behavior, status code, and impact.

## Phase 5: Generate a report

- Fill in an evidence JSON file.
- Run the report generator.
- Read the generated Markdown report.
