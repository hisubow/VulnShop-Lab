# Learning Path - Tiếng Việt

## Giai đoạn 1: Chạy lab

- Cài Docker Desktop.
- Chạy `docker compose up --build`.
- Mở hai app ở port 5000 và 5001.

## Giai đoạn 2: Khám phá web app

- Login bằng tài khoản demo.
- Thử Search, Feedback, Profile, Admin.
- Quan sát log trong Docker.

## Giai đoạn 3: Làm challenge

- Đọc `docs/vi/challenges.md`.
- Test từng case trên vulnerable app.
- Thử lại trên secure app.

## Giai đoạn 4: Ghi evidence

- Dùng `docs/vi/evidence-checklist.md`.
- Ghi endpoint, account, behavior, status code, impact.

## Giai đoạn 5: Sinh report

- Điền evidence JSON.
- Chạy report generator.
- Đọc report Markdown sinh ra.
